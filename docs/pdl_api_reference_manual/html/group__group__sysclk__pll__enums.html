PLL bypass mode.

See register CLK_PLL_CONFIG, bits BYPASS_SEL.

Enumerator

Output PLL input source when not locked, and PLL output when locked.

CY_SYSCLK_PLL_OUTPUT_AUTO1 

Same as AUTO.

CY_SYSCLK_PLL_OUTPUT_INPUT 

Output PLL input source regardless of lock status.

CY_SYSCLK_PLL_OUTPUT_OUTPUT
